Steps to locally setup development after cloning the project.

### Step 1 - Dependencies

You will need:

* [node](https://nodejs.org/)

### Step 2 - Install packages
run from the root of the repository:

```sh
npm install
```

### Step 3 - Run an app

Once the dependencies are installed, you can run the app:

```sh
cd kanban-board
# then
npm run dev
```
